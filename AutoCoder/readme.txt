## AutoCoder - Yapay Zekâ Destekli Kod Üretici

Bu proje, S4E DevOps staj teknik değerlendirmesi kapsamında geliştirilmiş basit bir AI destekli Python kod üretici uygulamasıdır. Teknofest yarışmasına katılım sağladığım ve zamanımın büyük kısmını bu yarışmaya ayırmam gerektiği için şu an projemin bazı kısımları eksik olsa da işlevsel bir versiyonunu hazırladım ve sizinle paylaşmak istiyorum. Eksiklerimi tamamlamak için elimden geleni yapacağım ve süreç içinde gelişim göstereceğimden şüpheniz olmasın.


🚀 Özellikler
Kullanıcıdan alınan prompt'a göre Python kodu üretir.

Üretilen kodla birlikte kısa ve anlamlı bir başlık oluşturur.

OpenAI Chat API ile entegre çalışır.

Flask tabanlı basit bir web arayüzüne sahiptir.

⚙️ Teknolojiler

Python

Flask

OpenAI API

HTML/CSS
